# AGENTS.md

Scope: the entire repository.

- Make granular, clean commits: one logical change per commit with a clear imperative message.
- Prefer open-source solutions; choose copyleft-compatible options when practical and document tradeoffs.
- Keep privacy as a default: minimize personal data collection and avoid tracking-heavy services.
- Scope ADRs to V1 unless a broader decision is explicitly requested.
- Apply SOLID and DRY pragmatically, following existing project patterns before adding abstractions.
- Keep imported archives, generated exports, local data, and temporary files out of git unless explicitly requested.
- Run the smallest relevant verification before committing a change, and report any skipped checks.
